# Quản lý Danh bạ (Contact Manager)

Ứng dụng Console quản lý liên hệ viết bằng Java, ứng dụng kiến thức OOP và ArrayList.

## Tính năng
- Thêm mới và hiển thị danh bạ.
- Bắt lỗi nhập liệu cơ bản (chống văng app).
- Tương tác qua giao diện dòng lệnh (Terminal/CMD).

## Hướng dẫn chạy nhanh

1. **Yêu cầu:** Máy đã cài **Java 8+** (kiểm tra bằng lệnh `java -version`).
2. **Tải file:** Tải `git_exercise.jar` tại mục **[Releases](../../releases)**.
3. **Chạy ứng dụng:** Mở Terminal/CMD tại thư mục chứa file và gõ lệnh:

   java -jar git_exercise.jar
